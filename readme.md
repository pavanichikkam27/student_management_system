# Student Management System (Python + MySQL)
This is a simple Student Management System built using Python and MySQL.
It is a console-based project that allows the user to manage student records in a database.

## Features
* Login system
* Add student details
* View all students
* Search student by roll number
* Update student details
* Delete student record
* Data stored in MySQL database

## Technologies Used
* Python
* MySQL
* mysql-connector-python

## How to Run the Project
1. Install Python

2. Install MySQL

3. Install mysql connector:
   pip install mysql-connector-python

4. Create database in MySQL

   * Database name: student_db
   * Table name: students
   * Table name: users (for login)

5. Run the project:
   student_management_mysql.py

## Database Structure
**students table**

* roll
* name
* age
* grade

**users table**

* username
* password
